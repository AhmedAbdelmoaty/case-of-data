## 🎮 ملخص اللعبة كاملة (المنطق المعتمد)

### القضية
أبو سعيد، صاحب محل ملابس. الموسم ده باع **أقل بـ20%** من السنة اللي فاتت. عايز يعمل خصومات. **الحقيقة المخفية**: السنة اللي فاتت كانت استثنائية (افتتاح مول جنبه جاب حركة). الأرقام دلوقتي طبيعية. **الدرس**: "Compared to what?" — مقارن بإيه؟

### الميكانيكا
- **20 سؤال** في البنك، اللاعب يختار **7 بس**
- **5 ذهبية** (تكشف الـ baseline) + **6 مضللة** + **4 قفز للحل** + **3 محايدة** + **2 تحدي إطار**
- **ثقة من 0 لـ 8**، تبدأ بـ 5، أسئلة قفز الحل تخصم منها
- **22 دليل** يتفتحوا حسب الأسئلة
- **التأطير**: اختار 1 عَرَض + 1 سبب + 1 توصية = جملة كاملة

### الـ 5 مسارات (موصوفة بالكامل في plan.md)
1. **🥇 المسار الذهبي** (Master Framer): سأل تحدي إطار + اكتشف baseline → "ما فيش مشكلة، المرجع غلط"
2. **🥈 المسار الفضي** (Solid Analyst): اكتشف الحقيقة بس التوصية ضعيفة
3. **🥉 المسار البرونزي** (Promising): اقترب بس ما حسمش
4. **❌ فخ Baseline متجاهَل**: عمل خصومات (الفخ الكلاسيكي)
5. **❌ فخ السبب الداخلي**: اتهم البائعين/الموقع
6. **💀 تأطير غير متماسك**: مكونات مش مترابطة

### تدفق الشاشات
```
Briefing → Travel → Arrival → InvestigationHub (الاستقصاء)
   → Synthesis (مراجعة الأدلة) → FramingBuilder (بناء الجملة) 
   → Debrief (النتيجة + الدرس)
```

## 📋 خطة المرحلة الجاية (المرحلة 4 — بناء كل الشاشات)

سأبني الـ 4 شاشات الناقصة + أربط كل حاجة. تقسيم مقترح في **3 جلسات متتالية**:

### الجلسة 1 — InvestigationHub (القلب)
- Header: ميزانية + ثقة (شريط) + عداد الأدلة
- قائمة 20 سؤال **بترتيب عشوائي ثابت**، بدون أي أيقونات تكشف النوع
- لما اللاعب يختار سؤال → بطاقة رد أبو سعيد + ظهور الأدلة الجديدة بـ animation
- لوحة جانبية للأدلة المكتشفة
- زر "خلاص — حلل" يظهر بعد 4 أسئلة على الأقل

### الجلسة 2 — Synthesis + FramingBuilder
- **Synthesis**: عرض الأدلة كبطاقات، اللاعب يثبّت اللي يعتبره مهم (pin)
- **FramingBuilder**: 3 أعمدة (5 أعراض / 6 أسباب / 6 توصيات). جملة معاينة حية تحت. زر تأكيد

### الجلسة 3 — Debrief + الربط
- **Debrief**: 
  - Hero: Tier (مع لون) + درجة من 100
  - 5 شرائط للمحاور (تحدي إطار، اكتشاف، اقتصاد، تركيب، ثقة)
  - كشف الحقيقة الكاملة
  - سجل الأسئلة (replay)
  - الدرس المكنزي
- ربط Briefing → Travel → Arrival → Hub بالشاشات الموجودة أصلاً
- زر "ابدأ من جديد" + حفظ النتيجة
